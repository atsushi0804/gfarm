gfmpirun

Name

gfmpirun -- run Gfarm parallel programs with mpirun

Synopsis

gfmpirun command args...

DESCRIPTION

gfmpirun executes a parallel command on filesystem nodes with mpirun.

Command can be specified by a gfarm-URL that is registered using gfreg(1).

BUGS

gfmpirun can only work with the MPICH-p4.
