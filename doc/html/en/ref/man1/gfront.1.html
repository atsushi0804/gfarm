gfront

Name

gfront -- Gfarm file system browser

Synopsis

gfront

DESCRIPTION

GFront is a GUI application to operate a gfarm filie system. GFront makes it possible that a user operates a gfarm file system interactively. Each operation for gfarm file system is already implemented as commands, and provide a front end function to execute a function equivalent to commands of gfarm in GFront.
