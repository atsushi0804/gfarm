gfs_rename

Name

gfs_rename -- change the name or location of a file

Synopsis

#include <gfarm/gfarm.h>

char *gfs_rename(char * gfarm_url_from, char * gfarm_url_to);

DESCRIPTION

gfs_rename renames gfarm_url_from to gfarm_url_to.

RETURN VALUES

NULL

The function terminated successfully.

GFARM_ERR_NO_MEMORY

Insufficient memory was available.

GFARM_ERR_AUTHENTICATION

User authentication failed when connecting to gfmd(8).

GFARM_ERR_PERMISSION_DENIED

gfarm_url_from or gfarm_url_from points outside your accessible address space.

GFARM_ERR_ALREADY_EXISTS

gfarm_url_to already exists. This includes the case where gfarm_url is a symbolic link, dangling or not.

GFARM_ERR_INVALID_ARGUMENT

gfarm_url_to contained a path prefix of gfarm_url_from, more generally, an attempt was made to make a directory a subdirectory of itself.

GFARM_ERR_NO_SUCH_OBJECT

gfarm_url_from does not exist.

Others

An error except the above occurred. The reason is shown by its pointed strings.
