gfarm_url_section_replicate_from_to

Name

gfarm_url_section_replicate_from_to -- 各フラグメントのレプリカを作成する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_url_section_replicate_from_to(char *gfarm_url, char *section, char *srchost, char *dsthost);

DESCRIPTION

srchostに存在する、 ファイル gfarm_url のフラグメント section のレプリカを、 dsthost に作成します。

この関数では、フラグメント番号を整数ではなく、文字列 section で指定する必要があることに注意してください。また、またこの文字列は、 余分な空白や、先行 0 を含んではいけません。

指定した srchost レプリカが存在しなかった場合にはエラーになります。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
