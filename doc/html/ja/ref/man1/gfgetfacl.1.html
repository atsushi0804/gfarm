Name

gfgetfacl — Gfarm ファイルのアクセス制御リストを表示する

Synopsis

gfgetfacl [オプション] [path...]

DESCRIPTION

gfgetfacl は Gfarm ファイルシステム上の ファイルやディレクトリの名前，所有者名，グループ名， アクセスコントロールリスト (ACL) を表示します．

OPTIONS

-R

再帰的にACLを表示します．

-?

引数オプションを表示します．

NOTES

Gfarm ファイルシステムから Gfarm のユーザやグループを削除しても、ACL エ ントリに指定したユーザ名やグループ名は ACL に残ります．

SEE ALSO

gfsetfacl(1)
