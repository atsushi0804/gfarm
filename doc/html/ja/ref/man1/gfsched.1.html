gfsched

Name

gfsched -- ファイル・アフィニティ・スケジューリングにより ホストリストを作成

Synopsis

gfsched [-I フラグメント番号] gfarm-URL [output-hostfile]

DESCRIPTION

指定した gfarm-URL を利用してファイル・アフィニティ・スケジュー リングによりホストリストを作成します。 output-hostfile が省略された場合、標準出力にホストリストを表示し ます。

OPTIONS

-I フラグメント番号

特定のファイル断片についてホストリストを作成します。 実行形式のファイルの場合には、このオプションで実行アーキ テクチャ(例 「sparc-sun-solaris」) を指定することによっ て、指定したアーキテクチャのホストリストを作成できます。

-?

引数オプションを表示します。
