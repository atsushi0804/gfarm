gfmpirun_p4

Name

gfmpirun_p4 -- mpirun コマンドの Gfarm 用フロントエンド

Synopsis

gfmpirun_p4 プログラム名 引数...

DESCRIPTION

引数に指定した Gfarm入力ファイル の存在する計算プールのホスト群に対して、 mpirun を実行します。

プログラム名としては、gfreg コマンドで登録したプログラムの gfarm-URL も指定できます。

BUGS

入力ファイルか否かは、単に、指定された Gfarm-URL が既に存在するかど うかだけで判断するので、Gfarm-URL の綴りを間違えると、意図したホスト で実行されません。

現バージョンは、MPI-CH/p4 との組合せでないと動作しません。
