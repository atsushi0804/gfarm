gfrep

Name

gfrep -- Gfarm ファイルのレプリカを作成する。

Synopsis

gfrep -H ホスト指定ファイル gfarm-URL

gfrep -D ドメイン名 gfarm-URL

gfrep -I フラグメント番号 [-s 複製元ホスト] -d 複製先ホスト gfarm-URL

DESCRIPTION

引数 Gfarm-URL の複製を、ホスト指定ファイルに記述されたホス トあるいは指定されたドメイン名のホストに作成します。

-I オプションでフラグメント番号あるいはセクション名を指定し た場合は、指定したファイルフラグメントの複製を -d オプション で指定したホストに作成します。-s オプションが指定された場合 は、指定されたホストからファイルの複製を作成します。引数 Gfarm-URL が ファイルフラグメントを一つしか持たない場合は、-I オプションによるフラ グメント番号の指定を省略することができます。

OPTIONS

-H ホスト指定ファイル

各フラグメントの複製を作成するホスト名のリストを指定します。 ホスト名は、各行に一ホストずつ記述します。ホストファイル中 のホスト名の数は、フラグメントの数と一致している必要があり ます。

ファイル名として ``-'' を指定した場合、標準入力からホス ト名のリストを読み込みます。

-D ドメイン名

各フラグメントの複製を作成するホストのドメイン名を指定します。

-I フラグメント番号

複製を作成するファイルフラグメントを指定します。 ファイルフラグメントはフラグメント番号あるいはアーキテク チャなどのセクション名です．

-s 複製元ホスト

-I オプションと共に指定し，ファイルフラグメントの複製元 のホストを指定します．

-d 複製先ホスト

-I オプションと共に指定し，ファイルフラグメントの複製先 のホストを指定します．

-?

引数オプションを表示します。
