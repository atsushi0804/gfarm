gfrm

Name

gfrm -- Gfarm ファイルを削除する。

Synopsis

gfrm [-h ファイルシステムノード] gfarm-URL...

gfrm -h ファイルシステムノード -I フラグメント番号 gfarm-URL...

DESCRIPTION

Gfarm-URL に指定したファイルを削除します。-h オプションで ファイルシステムノードを指定することにより、特定のファイル システムノードに存在するファイル断片の複製を削除することが できます。

また、 -h オプションと -I オプションの両方を指定すること により、特定のファイル断片の複製を削除することができます。 ただし、複製が一つしかない場合は削除することができません。

OPTIONS

-h ファイルシステムノード

複製を削除するファイルシステムノード名を指定します。 -I オプションと共に指定することにより、特定のファイル断片を 削除することができます。複数のファイルシステムノードか ら同じファイル断片の複製を一度に削除するためには，このオ プションを複数回指定します．

-I フラグメント番号

複製を削除するフラグメント番号を指定します。 実行形式のファイルの場合には、このオプションで実行アーキ テクチャ(例 「sparc-sun-solaris8」) を指定することによっ て、指定したアーキテクチャの複製を削除できます。

-?

引数オプションを表示します。
