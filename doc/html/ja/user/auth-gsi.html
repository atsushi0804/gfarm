GSI 認証設定

English | 日本語

この文書は、Gfarm の GSI 認証で必要な設定について説明します。

config-gfarm

以下のように -a オプションで gsi ないし gsi_auth を指定して初期設定します． 
# config-gfarm -t -A >global_admin_name< -a gsi_auth -D >Subject DN<
-D オプションでは Gfarm ファイルし捨て無管理者の X.509 クライアント証明書の Subject Name を指定します。

指定認証方式に関する詳細は gfarm2.conf(5) の マニュアルページあるいは Gfarm-FAQ.ja を参照してください．

もしデフォルト値が適切でない場合には，表示されるオプションを指定して設 定を変更できます．適切な設定になったら，config-gfarm コマンドのオプショ ンから -t を取り除いて実行します．

GSI認証を用いるためには，ホスト証明書やクライアント証明書を用意した上で 各種設定を行なう必要がありますが，ここではその説明を省略しますので 「 GSI C Admin Guide」を参照して下さい．
